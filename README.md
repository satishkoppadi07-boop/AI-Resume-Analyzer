# AI Resume Analyzer

AI Resume Analyzer is a web application that analyzes resumes and provides ATS scores, candidate level assessment, and AI-powered career insights.

## Features

- Resume Upload
- ATS Score Calculation
- Candidate Level Detection
- AI Career Insights
- PDF Report Download
- User Friendly Interface

## Technologies Used

- Java
- Spring Boot
- Thymeleaf
- HTML
- CSS
- Bootstrap
- Google Gemini API
